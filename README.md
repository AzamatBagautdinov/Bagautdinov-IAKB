# Bagautdinov-IAKB
Багаудинов Азамат
Описание кода
Представленный код на Python реализует приложение‑генератор случайных задач с графическим интерфейсом (GUI) на базе библиотеки tkinter
Сценарий 1. Генерация случайной задачи

Запустите приложение.

Нажмите кнопку «Сгенерировать задачу».

В верхней части окна появится случайная задача из списка PREDEFINED_TASKS (например, «Написать отчёт по выполненной работе»).

Эта задача автоматически добавится в историю (отображается в списке внизу окна) и будет сохранена в файл tasks.json.

Сценарий 2. Добавление новой задачи

Введите название новой задачи в поле «Добавить задачу» (например, «Провести исследование»).

Выберите категорию задачи из выпадающего меню (например, «работа»).

Нажмите кнопку «Добавить задачу».

Новая задача добавится в список предопределённых задач, появится в истории и будет сохранена. При последующих нажатиях кнопки «Сгенерировать задачу» она может быть выбрана случайным образом.

Сценарий 3. Фильтрация истории по категории

Выберите категорию в выпадающем меню «Фильтр по типу» (например, «учёба»).

Список истории задач обновится и покажет только те задачи, которые относятся к выбранной категории (например, «Решить задачу», «Написать ЭССЕ»).

Выберите «все», чтобы снова увидеть полную историю.

Сценарий 4. Сохранение и загрузка истории

Сгенерируйте или добавьте несколько задач.

Закройте приложение (просто закройте окно).

Снова запустите приложение.

Вы увидите, что история задач, сохранённая в файле tasks.json, автоматически загрузилась и отображается в списке истории.
